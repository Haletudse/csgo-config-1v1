# 1v1 CS:GO Config
My 1on1 config for Counter-Strike: Global Offensive.

### Execute
```
exec 1v1
```
### Commands
Starting a default 1v1 match.
```
startgame
startmatch
```
Restart the 1v1 match.
```
restartgame
restartmatch
```
Overtime

Command toggles on & off
```
overtime
```
Allow Weapons placed in the map

Command toggles on & off
```
groundweapons
mapweapons
```

Headshot only

Command toggles on & off
```
headshot
```

Pause & Unpause match on round start/end.
```
pausegame
pausematch
unpausegame
unpausematch
```
